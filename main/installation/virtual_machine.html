Virtual Machine

Attention

Skip this section if you do not want to use a virtual machine but you have installed Ubuntu directly on your computer (recommended). Continue with the installation of ROS.

For instructions on how to install Ubuntu in a Virtual Machine please use your preferred search engine to find one of the many tutorials out there.

For performance reasons we recommend to run Ubuntu alongside Windows as Dual Boot as described in the Install Ubuntu section.

Troubleshooting

If you can not launch the gazebo command after you have installed ros-noetic-desktop-full in the section Install ROS please execute the command

echo "export SVGA_VGPU10=0" >> ~/.profile

and restart your Virtual Machine.
